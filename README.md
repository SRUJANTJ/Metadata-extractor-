File  Metadata Extractor

Supported File Formats

JPEG (.jpg, .jpeg)
PNG (.png)
TIFF (.tiff)
BMP (.bmp)
GIF (.gif)
WebP (.webp)
HEIC (.heic)
                                                                        
Installation
                                                                      
Download the latest release from GitHub Releases.
Extract the downloaded zip file.
Run ImageMetadataExtractor.exe (No installation required).
